Dépendances

  • boost
  • OpenGL >= 3.3
  • SDL2
  • GLU
  • GLEW
  • SDL2_image (included)
  • GLM
  • Scons (for building)

Installation

Pour lancer le projet, exécuter sur votre console ces deux commandes : 

scons
./solarsystem
